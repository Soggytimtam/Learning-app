# Peppol Quest art direction

## Visual goal
Create an original cozy-adventure visual language that feels:
- miniature
- playful
- handcrafted
- bright but soft
- storybook technical
- readable for beginners

This should feel like a tiny explorable learning world, not a SaaS dashboard.

## Mood words
- mossy
- glowing
- parchment
- toy-town
- lantern-lit
- friendly
- curious
- gem-like
- tactile
- layered

## Core visual pillars

### 1. Miniature overworld feeling
The player should feel like they are looking at a tiny crafted world on a tabletop.
Use:
- rounded terrain tiles
- little paths
- puffball trees
- chunky buildings
- floating badge markers
- layered shadows

### 2. Warm teaching energy
The world must feel safe for beginners.
Use:
- soft cream and parchment panels
- warm green and gold highlights
- rounded corners everywhere
- gentle gradients
- friendly iconography

### 3. Technical magic
Peppol concepts should feel like systems magic.
Use:
- glowing runes for identifiers
- crystal/portal motifs for discovery
- harbor lights and signal beacons for AS4
- tribunal seals for rules and validation
- map scrolls and notebooks for BA deliverables

## Character shape language
- oversized heads slightly optional, but not chibi parody
- rounded silhouettes
- simple readable outfits
- distinctive hats/capes/tools tied to each role
- low-detail faces with strong expression readability

## UI rules
- never use sharp corporate rectangles if a rounded panel can work
- every panel should feel like a crafted object
- use layered cards with soft shadow and subtle highlight edges
- icons should be chunky and slightly whimsical
- progress bars should feel like filling lanterns, leaves, or crystals

## Animation feel
- tiny bobbing motion
- subtle shimmer on important interactables
- floating particles sparingly
- gentle hover lift
- no hyperactive motion

## What to avoid
- exact imitation of Nintendo compositions or assets
- generic fintech dashboard visuals
- neon cyberpunk overload
- too much realism
- visually noisy data tables as the main view

## Asset priorities
1. world tiles
2. buildings and landmarks
3. quest icons
4. NPC portraits
5. collectible badge/emblem set
6. glossary symbol set
7. decorative UI borders and dividers
