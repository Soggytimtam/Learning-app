# UI screen blueprints

## 1. Home / World Map
### Above the fold
- oversized logo lockup
- world subtitle in beginner-friendly language
- one large primary CTA: Start journey
- one secondary CTA: Continue as analyst
- floating badge strip for BA / Technical BA / A-NZ Specialist tracks

### Main map area
- chunky illustrated region cards or tiles
- visible progression roads between zones
- sparkles or glow on the next recommended zone
- small corner panel showing glossary word of the day

## 2. Module detail screen
- top banner art for zone
- left column for story and lesson text
- right column for quest log, glossary popovers, and rewards
- toggle between Plain English and Technical View
- big visual deliverable card at bottom

## 3. Incident Detective screen
- evidence board layout
- NPC statements pinned like cards
- route trace panel
- likely root-cause selector
- recommendation composer with audience toggle: exec / finance / developer

## 4. Glossary Codex
- searchable symbol grid
- each term shown with icon, plain explanation, technical explanation, BA relevance
- unlock animation when first encountered in gameplay

## 5. Skill Tree Hall
- giant branching skill tree
- each node represented by gem, scroll, lantern, or compass motif
- branches: BA, Technical BA, A-NZ Specialist, Strategy
- hover reveals practical job capability unlocked

## Visual polish checklist
- every screen must have one focal illustration area
- every panel should feel collectible or crafted
- avoid flat enterprise layout monotony
- always show progression and wonder
